Viola is a lightweight web proxy designed to provide fast and reliable web access in restricted environments. It is primarily developed for integration
with the COSMOSIS platform, allowing users to access web resources while maintaining a simple and flexible deployment structure. Viola acts as a web 
proxy that retrieves and serves web content through a controlled gateway. It is designed to be minimal, adaptable, and easy to deploy on a variety 
of hosting platforms. While Viola is primarily built to support COSMOSIS, it can also be used as a standalone proxy for development, research, 
or educational purposes.
